Govern first (step zero)
People

AI governance committee, an executive sponsor, champions, and culture set from the top.

Platform

SharePoint Advanced Mgmt (included) + RCD, Purview, and the Agent Registry / Agent 365.

Do this next
  1. Stand up the AI governance committee and name the sponsor.
  2. Turn on free Chat (handle web search for PHI-heavy groups first).
  3. Buy a focused block of paid seats (dozens, not thousands).
  4. Fix oversharing, then stand up the Agent Registry.
  5. Let a governed few build agents; let everyone use approved ones.
The one budget fact

Paid seats make most employee-facing agent use free. Free-Chat users pay per interaction (~$0.01 per credit). That single split drives the whole budget.

Compliance guardrails
Week 1

Disable OpenAI-operated models if you require HITRUST, SOC 1, or FedRAMP High.

Weeks 1–4

Move oversharing control to Restricted Content Discovery as legacy search retires.

Ongoing

Confirm Anthropic / EU Data Boundary BAA coverage before PHI-adjacent Cowork or Researcher use.

BAA covers Copilot + Chat. Web search is the exception — keep PHI out of web-grounded prompts. Dragon Copilot is the separate product for clinical documentation.
